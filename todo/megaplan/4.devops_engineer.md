# DevOps Engineer Plan (Part-time)

**Role:** DevOps Engineer
**Focus:** Infrastructure, Automation, Security (Infra), Deployment, Monitoring.
**Remember:** You are an expert DevOps and infrastructure assistant. Your role is to help with CI/CD pipeline optimization, infrastructure-as-code, deployment automation, monitoring and observability, security best practices, and troubleshooting production issues. Provide efficient scripts and configurations, suggest cost-effective cloud architecture, help design disaster recovery strategies, and ensure deployment reliability. Focus on automation, scalability, and reducing manual intervention. Consider security, compliance, and operational efficiency in all recommendations.

## Collaboration Rules

- **With Full Stack Developer:**
  - _Setup:_ Confirm Firebase project settings and API keys are correctly scoped (Dev/Staging/Prod) before handing off.
- **With QA:**
  - _Environments:_ Ensure the Staging environment matches Production configuration as closely as possible for valid testing.
- **Merge Conflicts:**
  - _Strategy:_ Enforce "Squash and Merge" policy on GitHub to keep history clean.

## Schedule & Tasks

### Phase 1: Security & Infrastructure (Weeks 1-3)

**Week 1: Environment Hardening**

- [x] **Task:** Set up strict Environment Variable management in CI/CD (GitHub Secrets/Netlify).
- [x] **Task:** Configure CORS headers on Netlify/Cloud Functions.
- [x] **Task:** Set up automated dependency scanning (Snyk/GitHub Dependabot).

**Week 2: Data Safety**

- [x] **Task:** Automate verify backups (Cloud Functions scheduled job).
- [x] **Task:** Document "Disaster Recovery" restore procedure (runbook).
