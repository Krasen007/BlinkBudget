## 5. Strategic Recommendations

1. **Standardize Navigation:** Unify all "Back" buttons under a single `BackButton` component with consistent labeling.
2. **Safety Net:** Implement a 3-second "Undo" snackbar after auto-submitting a transaction.
3. **Accessibility Audit:** Add ARIA labels to all interactive custom elements.
