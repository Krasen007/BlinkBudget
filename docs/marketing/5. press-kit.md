## Project Description

BlinkBudget is an open-source expense tracking application designed to make budgeting effortless and habitual. The app focuses on fast, simple expense entry with a 3-click workflow and comprehensive financial analytics.

## Key Features

### Core Functionality

- **3-Click Expense Entry**: Amount -> Category -> Done workflow
- **Auto-Submit**: Transactions save automatically on category selection
- **Multiple Transaction Types**: Expenses, Income, Transfers, Refunds
- **Multi-Account Support**: Checking, Savings, Credit Card, Cash accounts
- **Custom Categories**: Create, edit, and manage personalized expense categories
- **Transaction History**: Full edit/delete capabilities with date and category filtering

### Analytics & Planning

- **Real-time Balance Calculations**: Across all accounts
- **Financial Planning Engine**: Long-term goal planning and projections
- **Investment Portfolio Tracking**: Manual entry for various investment types
- **Anomaly Detection**: Identifies unusual spending patterns
- **Advanced Analytics**: Top movers analysis, timeline comparisons, trend insights
- **Dynamic Charting**: Daily and monthly view toggles

### User Experience

- **Progressive Web App (PWA)**: Installable with offline support
- **Mobile-Optimized**: Touch-friendly interface with 56px minimum touch targets
- **Responsive Design**: Works on desktop and mobile devices
- **Offline Capable**: Full functionality without internet connection using localStorage
- **Keyboard Shortcuts**: Escape key navigation support

### Data Management

- **Local-First Storage**: Primary data stored in browser localStorage
- **Firebase Sync**: Optional cloud synchronization across devices
- **Daily Backups**: Automatic daily backup to Firebase
- **Data Export**: Export financial data in various formats
- **Restore Functionality**: Restore from last backup with user warnings

## Technical Specifications

### Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+ Modules) + Vite
- **Styling**: Vanilla CSS with CSS Custom Properties and PostCSS
- **Storage**: Firebase (cloud sync) + localStorage (offline)
- **Charts**: Chart.js for data visualization
- **Testing**: Vitest with jsdom environment
- **Build Tools**: Vite with PostCSS optimization pipeline
- **PWA**: vite-plugin-pwa for service worker management

### Architecture

- **Component-Based**: Functional components returning native HTMLElements
- **Modular Structure**: Organized into components, views, core services, utils, and styles
- **Performance Optimized**: CSS purging, asset optimization, intelligent caching
- **Security Focused**: XSS protection, URL validation, privacy compliance

### Performance Features

- **Fast Chart Loading**: Intelligent caching system with cache invalidation
- **Optimized Builds**: CSS minification, unused style removal
- **Bundle Size**: Optimized production builds
- **Offline Support**: Service worker for offline functionality
