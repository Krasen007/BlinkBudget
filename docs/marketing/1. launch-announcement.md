## Project Overview

BlinkBudget is an open-source expense tracking application designed to make budgeting effortless and habitual. The app focuses on fast, simple expense entry with a 3-click workflow and comprehensive financial analytics.

## Core Features

### Fast Expense Entry

- **3-Click Entry**: Amount -> Category -> Save workflow
- **Auto-Submit**: Transactions save automatically on category selection
- **Mobile Optimized**: Perfect for one-handed, on-the-go use

### Data Management

- **Multiple Transaction Types**: Expenses, Income, Transfers, Refunds
- **Custom Categories**: Create, edit, and manage personalized expense categories
- **Multi-Account Support**: Checking, Savings, Credit Card, Cash accounts
- **Transaction History**: Full edit/delete capabilities with filtering

### Analytics & Planning

- **Real-time Balance Calculations**: Across all accounts
- **Financial Planning Engine**: Long-term goal planning and projections
- **Investment Portfolio Tracking**: Manual entry for various investment types
- **Advanced Analytics**: Spending patterns, trends, and insights

### User Experience

- **Progressive Web App (PWA)**: Installable with offline support
- **Responsive Design**: Works on desktop and mobile devices
- **Offline Capable**: Full functionality without internet connection
- **Privacy First**: Local data storage with optional cloud sync

## Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+ Modules) + Vite
- **Styling**: Vanilla CSS with CSS Custom Properties and PostCSS
- **Storage**: Firebase (cloud sync) + localStorage (offline)
- **Charts**: Chart.js for data visualization
- **Build**: Vite with PostCSS optimization pipeline

## Access & Installation

### Web Application

- **Live App**: https://blinkbudget.netlify.app
- **Open Source**: https://github.com/Krasen007/BlinkBudget
- **PWA Support**: Installable on any modern browser

### Android Application

- **APK Download**: Available via GitHub releases
- **Trusted Web Activity**: Native Android app experience
- **Auto-Updating**: Updates automatically from web version

### Development Setup

```bash
# Clone the repository
git clone https://github.com/Krasen007/BlinkBudget.git
cd blinkbudget

# Install dependencies
yarn install

# Start development server
yarn run dev
```
