# Changelog

## Recent Updates

### Transaction Management
- **Fixed timestamp handling**: When editing a transaction without changing the date, the original time is now preserved instead of being reset to current time
- **Improved date/time consistency**: Ensured proper timestamp updates when adding or editing transactions

### Financial Planning & Reports
- **Fixed caching issues**: Resolved data caching problems in the Financial Planning section
- **Moved functionality**: ReportsView features integrated into Financial Planning Insights tab for better organization
- **Fixed chart rendering**: Corrected display issues with charts showing negative values
- **Fixed currency calculations**: Corrected currency and calculation errors in Financial Insights chart

### User Interface Improvements
- **Fixed Reports view navigation**: Improved navigation and filter preservation in Reports view
- **Added filter breadcrumbs**: Added "Currently filtered by" indicators that allow quick return to category view
- **Fixed Dashboard timing**: Resolved UTC method usage in Dashboard view for accurate date/time display
- **Fixed z-index issues**: Corrected layering in Spending by Category section
- **Improved button styling**: Standardized button appearance across the app
- **Fixed margins**: Corrected spacing issues in various sections

### Feature Removal
- **Removed Time of Day Patterns**: Removed this analytics feature as it was not providing useful insights

### Code Quality & Security
- **Snyk security fixes**: Applied security patches for dependency vulnerabilities
- **Code linting**: Improved code consistency and formatting
- **General optimizations**: Various performance improvements and bug fixes

### Version
- **Version 1.33**
