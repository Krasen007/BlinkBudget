## v1.38

### ✨ New Features
- **Quick Amount Presets toggle** — You can now show or hide the recent amount buttons on the Dashboard from Settings → Advanced. They're hidden by default for a cleaner look.
- **Minus sign auto-switches to Refund** — Typing `-` in the amount field instantly switches the transaction type to Refund, so logging a refund takes no extra steps.

### 🐛 Bug Fixes
- **Backup showed a fake time** (e.g. "3:00:00 AM") — the backup card now shows only the date, since no time is recorded.
- **Date format setting was ignored** in CSV exports, goal target dates, and filter summaries — all now follow your chosen format (US / EU / ISO).
- **Editing a transaction right after adding it no longer creates a ghost entry** — ghost transactions are only created for older edits, not immediate corrections.

### 🚀 Improvements
- **Mobile nav loads instantly** — returning users see the bottom navigation bar right away instead of waiting for the cloud sign-in check.
- **Dashboard balance label** changes to "Total Filtered" when a filter is active, so it's clear the number isn't your full balance.
- **Settings date format note** — a short note now explains that the date picker in the transaction form follows your browser/OS locale, not this setting.
