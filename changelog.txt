# Changelog - BlinkBudget

## Highlights
- **New Feature: Split Transactions** - You can now split a transaction into two by using a long-press on the dashboard.
- **UI Refinement** - Major overhaul of the Reports UI for better clarity. Optimized charts, tooltips, and mobile button layouts for a smoother experience.
- **Performance Boost** - Optimized PWA background behavior and reduced transaction handling delays for a more responsive feel.
- **Architecture Cleanup** - Improved codebase maintainability by separating storage logic, decomposing the analytics engine, and optimizing routing.
- **General Polish** - Fixed various CSS issues, resolved test failures, and performed a final codebase audit for production readiness.
